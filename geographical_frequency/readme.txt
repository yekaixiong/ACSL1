This directory contains the R scripts to draw the geographical frequency distribution of genetic variants. Multiple R packages need to be installed: maps, mapdata, mapplots, plotrix. 

Example command lines are:
$ Rscript scripts_for_modern_groups.rs56302210.RGB.R 
$ Rscript scripts_for_modern_groups.rs2046814.RGB.R  

Figures in pdf format will be exported. Example files are: "Frequencies_in_modern_groups.rs56302210.pdf" (Figure 1F) and "Frequencies_in_modern_groups.rs2046814.pdf" (Figure 1G).
