This folder contains scripts and input files for generating Figure 1A, 1B (from GTEx) and 1D, 1E (from eQTLGen).

$ Rscript draw_figure.all_tissues.eQTLs.Fig1.20240610.R  go 

This command line generates "ACSL1.Fig1.20240610.png", which is Figure 1A and 1B. 

$ Rscript draw_figure.eQTLGEN.R go   

This command line generates "ACSL1.eQTLGen.20240610.png", which is Figure 1D and 1E.
